# 🍔 Online Food Delivery System | Python, Django & MySQL Project  

**Online Food Delivery System** is a **full-stack Python Django project with MySQL database**.  
It manages **Food Items, Food Types, Companies, Customers, Orders, Items, and Order Status Updates**.  

This project features **two types of users**:  
- **Admin** → Manage food items, customers, orders, companies, and reports.  
- **Customer** → Register, browse food items, place orders, and track order history.  

This is a **major Django project** designed for **final year computer science students (B.Tech, MCA, BCA, M.Tech)** with **source code, database, and documentation**.  

---

## 📖 Abstract  

The **Online Food Delivery System** streamlines ordering and management of food delivery operations.  
- **Admin** → Perform CRUD operations on food items, customers, orders, and companies.  
- **Customer** → Browse food items, add to cart, place orders, and track them.  

It is an excellent choice for **Python Django MySQL final year projects**.

## 📂 Modules & Features  

### 👤 Customer Module  
- Admin can add, edit, update, and delete customers.  
- Customers can view/update their own profile.  
- Forms validated on client-side using JavaScript.  

### 🍕 Food Items Module  
- Admin manages food items (CRUD).  
- Customers can view available food items.  

### 🥗 Food Items Type Module  
- Admin manages item categories (CRUD).  
- Customers can view food item types.  

### 🏭 Food Company Module  
- Admin manages food companies.  
- Customers can view food item companies.  

### 🛒 Order Module  
- Admin can manage all orders.  
- Customers can view their orders and receipts.  

### 📦 Item Module  
- Admin manages items within orders.  
- Customers can view purchased items.  

### 📌 Order Status Update Module  
- Admin updates order statuses (CRUD).  
- Customers can track their orders.
